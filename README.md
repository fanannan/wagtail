# wagtail

Experimental online classifiers implemented in clojure.

## Usage

The following online classifiers are implenteed under the same framework.
  Mergin perceptron
  Passive agressive
  Confidence-weighted
  Soft confidence-weighted

Any suggestions are welcome.
